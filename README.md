Overview
========

This project is my homemade, Docker-deployed streaming ETL pipeline. It ingests real-time NYC subway data from the free online API ([https://api-endpoint.mta.info/Dataservice/mtagtfsfeeds](https://api.mta.info/#/subwayRealTimeFeeds)) using Apache Kafka and processes it in real-time with Apache Flink to deliver real-time analytics, trip tracking, and anomaly detection.

Programming Languages: Python, SQL

Technologies: VS Code, Apache Kafka, Apache Flink, Postgres (Database)
